CHAP.CHAP_Controller.Dynamixel.polar(x, y)[source]¶

Find distance to user input location in radians and hypotenuse.

@type x: int @param x: X-axis input from user control joystick (1.0 <-> -1.0).

@type y: int @param y: Y-axis input from user control joystick (1.0 <-> -1.0).

@rtype: float,float @return: mag (hypotenuse to destination, minimum value is 1) and ta (radians to destination)
